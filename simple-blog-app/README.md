# Simple Blog Demo Application

## Install

1. `composer install`

2. `cd src/ViewiApp/js`

3. `npm install`

## Run

### 1. Run NPM watch command:

   Root folder: `simple-blog-app`

   `cd src/ViewiApp/js`

   `npm run watch`

### 2. Run PHP dev server:

Root folder: `simple-blog-app`

`cd public`

`php -S localhost:8000`
